# Driver Web Application

## 🚀 TaxiLibre Driver Interface

React application for taxi drivers to manage rides, earnings, and profile.

### 🛠️ Tech Stack
- React 18
- Vite
- Tailwind CSS
- React Router
- Socket.io Client
- Axios

### 📦 Deployment
Deployed on Vercel: https://taxilibre-driver-ibhcha995-shopdstoks-projects.vercel.app

### 🔧 Environment Variables
- `VITE_API_URL`: Backend API endpoint

### 🚀 Features
- Real-time ride tracking
- Earnings dashboard
- Profile management
- Notifications system
- Document upload
